# Aicte_Project

This project analyzes Electric Vehicle data using AI and ML to identify adoption trends, top manufacturers, and electric range patterns. It predicts EV performance, explores regional growth, and provides insights to support sustainable transportation and data-driven decisions.



Electric Vehicle Range Prediction and Recommendation System. This project analyzes EV data, predicts electric range using ML models, and recommends suitable EVs based on user preferences. It includes data cleaning, EDA, model training, and a working Streamlit app for real-time predictions and recommendations.
